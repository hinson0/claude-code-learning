# 内容库

本目录用于存放正在创作或已完成的文章内容，按状态和类型归档。

## 目录结构

```
内容库/
├── README.md          # 本文件
├── 草稿/              # 正在创作中的文章
├── 已发布/            # 已发布的文章存档
└── 模板/              # 可复用的写作模板
```

## 文章状态流转

```
选题确认 → [草稿/] → 修改打磨 → [已发布/]
```

## 文件命名规范

```
YYYY-MM-DD_文章标题.md
示例：2026-03-03_费曼学习法的实操版本.md
```

## 文章元数据格式

每篇文章开头必须包含以下元数据块：

```markdown
---
title: 文章标题
date: YYYY-MM-DD
status: draft | review | published
category: 科技与AI | 个人成长 | 商业洞察 | 生活方式
tags: [标签1, 标签2]
word_count: 0
platform: 公众号 | 小红书 | 微博 | 知乎
---
```

## 质量检查清单

发布前必须完成以下检查：

- [ ] 标题是否足够吸引人（套用选题库中的标题公式）
- [ ] 开头50字是否能抓住读者
- [ ] 每段是否都有核心结论，且结论前置
- [ ] 是否有具体案例/数据支撑核心观点
- [ ] 结尾是否有明确的行动号召或思考引导
- [ ] 是否符合风格说明中的禁忌清单
- [ ] 全文通读，删掉所有废话段落
