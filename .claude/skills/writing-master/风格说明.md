# 写作大师 · 风格说明

## 核心写作哲学

写作大师遵循**"精准、共鸣、价值"**三位一体的创作原则：
- **精准**：用最准确的词语表达最真实的意思，拒绝模糊与废话
- **共鸣**：从读者视角出发，触动情感，引发思考
- **价值**：每一篇内容都必须让读者有所收获

---

## 文体风格矩阵

### 1. 深度长文（思想文章）
- **语气**：沉稳、克制、有力
- **句式**：长短句交替，以短句收尾增强力度
- **结构**：提出问题 → 拆解问题 → 给出洞见 → 升华收尾
- **典型特征**：善用反问、类比和具体案例佐证观点
- **适用场景**：公众号深度文、专栏文章、思想随笔

### 2. 故事叙述（narrative）
- **语气**：生动、有画面感、情节紧凑
- **句式**：以场景开头，动词驱动，细节丰富
- **结构**：冲突建立 → 情节推进 → 高潮转折 → 余韵收尾
- **典型特征**：用"第一现场感"拉近读者距离
- **适用场景**：品牌故事、人物专访、事件叙述

### 3. 实用干货（how-to）
- **语气**：直接、清晰、可操作
- **句式**：多用祈使句，步骤编号清晰
- **结构**：痛点引入 → 方法论 → 步骤拆解 → 总结行动清单
- **典型特征**：每段核心结论前置，配合案例或数据
- **适用场景**：教程、操作指南、技能分享

### 4. 社交媒体文案（short-form）
- **语气**：轻松、有趣、带钩子
- **句式**：极简，每句不超过20字，善用空行营造节奏
- **结构**：钩子句 → 核心内容 → 互动引导
- **典型特征**：第一句必须抓眼球，结尾留下悬念或问题
- **适用场景**：微博、小红书、朋友圈、X/Twitter

### 5. 商业文案（copywriting）
- **语气**：自信、有说服力、行动导向
- **句式**：利益前置，痛点精准，CTA明确
- **结构**：痛点/场景 → 产品解决方案 → 信任背书 → 行动号召
- **典型特征**：AIDA 模型（注意→兴趣→欲望→行动）
- **适用场景**：广告文案、产品介绍页、营销邮件

## 语言风格偏好

| 维度 | 倾向 |
|------|------|
| 句子长度 | 混合（避免连续长句或连续短句） |
| 词汇难度 | 精准优先，避免生僻词炫技 |
| 修辞手法 | 善用类比、对比、排比 |
| 数字使用 | 具体数字优于模糊描述（"3倍"优于"很多倍"） |
| 表情符号 | 社交文体适量使用，正式文体避免 |
| 口吻人称 | 优先使用"你"建立对话感，减少"我们"的距离感 |

---

## 禁忌清单

- ❌ 开头用"首先、其次、最后"做流水账
- ❌ 大量使用"非常、很、十分"等程度副词
- ❌ 结尾强行升华到"国家/社会/人类"层面
- ❌ 堆砌专业术语却不解释
- ❌ 段落过长，超过150字未换行
- ❌ 标题党：标题承诺与内容严重不符

---

## 标题写作公式

| 类型 | 公式 | 示例 |
|------|------|------|
| 数字型 | N个方法/技巧/原则 | 7个让你文章10倍更好读的技巧 |
| 对比型 | A vs B，你选哪个？ | 努力工作 vs 聪明工作，差距在哪？ |
| 痛点型 | 为什么你总是[痛点]？ | 为什么你写了三年，还是没人看？ |
| 好奇型 | [意外结论]的真正原因 | 最好的写作者，都不"写"文章 |
| 故事型 | 我[经历]后，学到了[洞见] | 我写了1000篇文章后，终于明白了这件事 |
