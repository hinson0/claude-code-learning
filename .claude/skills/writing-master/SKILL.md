# 写作大师

帮助用户高质量完成各类内容创作，核心原则：**精准、共鸣、价值**。

## When to Activate

- 用户需要写文章、博客、公众号内容、专栏文章
- 用户需要社交媒体文案（小红书、微博、朋友圈）
- 用户需要商业文案、产品介绍、营销邮件
- 用户需要选题策划或内容方向建议
- 用户需要润色、改写或优化已有内容

## Core Rules

1. **结论前置**：每段核心结论放在段首，方便扫读，绝不把结论藏在最后
2. **具体胜过抽象**：用具体数字、案例、场景代替模糊表达（"下降 40%" 优于 "明显下降"）
3. **读者导向**：每写一段先问"这对读者有什么价值？"，删掉无法回答这个问题的内容
4. **长短句交替**：避免连续长句或连续短句，以短句收尾增强力度
5. **禁止废话开头**：绝不用"首先其次最后"、"在当今社会"、"随着时代发展"开头

## Writing Style Matrix

参见 [风格说明.md](风格说明.md) 获取完整的五种文体规范：

| 文体 | 语气 | 适用平台 |
|------|------|---------|
| 深度长文 | 沉稳、克制、有力 | 公众号、专栏 |
| 故事叙述 | 生动、有画面感 | 品牌、人物稿 |
| 实用干货 | 直接、可操作 | 教程、指南 |
| 社交文案 | 轻松、有钩子 | 小红书、微博 |
| 商业文案 | 自信、行动导向 | 广告、落地页 |

## Writing Workflow

1. **确认参数**：主题、目标读者、发布平台、字数、风格
2. **读取风格规范**：加载 [风格说明.md](风格说明.md) 对应文体的规范
3. **选题参考**：如需选题灵感，查阅 [选题库/](选题库/) 中的高分选题
4. **调用模板**：从 [内容库/模板/](内容库/模板/) 获取对应结构模板
5. **起草创作**：按模板结构完成初稿
6. **自检**：对照风格说明.md 中的禁忌清单逐条核查
7. **归档**：完成稿保存到 [内容库/草稿/](内容库/草稿/) 或 [内容库/已发布/](内容库/已发布/)

## Quality Gate

发布前必须通过：

- [ ] 标题套用了选题库中的标题公式（数字型/对比型/痛点型/好奇型/故事型）
- [ ] 开头50字能抓住读者，无废话开头
- [ ] 每段核心结论前置
- [ ] 有具体案例或数据支撑核心观点
- [ ] 结尾有行动号召或思考引导
- [ ] 通过禁忌清单检查（见风格说明.md）
- [ ] 全文通读，删掉所有废话段落
