# 选题库

本目录用于存放写作选题，按类别归档，方便快速查找与激发灵感。

## 目录结构

```
选题库/
├── README.md          # 本文件
├── 科技与AI.md        # 科技、人工智能相关选题
├── 个人成长.md        # 成长、学习、思维相关选题
├── 商业洞察.md        # 商业、创业、职场相关选题
└── 生活方式.md        # 生活、健康、文化相关选题
```

## 选题评分标准

每个选题按以下维度打分（1-5分）：

| 维度 | 说明 |
|------|------|
| 时效性 | 是否契合当前热点或趋势 |
| 读者价值 | 读者能从中获得多少实际收益 |
| 差异化 | 是否有独特视角，区别于已有内容 |
| 可操作性 | 是否容易落地执行（写作难度适中） |

**综合分 ≥ 16 分**的选题优先创作。

## 使用方式

1. 浏览各分类文件，找到感兴趣的选题
2. 参考选题的「角度建议」确定写作切入点
3. 在「内容库」中创建对应的草稿文件
4. 完成后在选题旁标注 `[已创作]` 并附上文件链接
